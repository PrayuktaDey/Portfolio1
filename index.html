Virtual Harmonics Analyser

In this project, the aim is analysing harmonic components present in any non-sinusoidal waveform, from a non-liner circuit(simulated in Simulink) by using an analyser. We performed the analysis using FFT and Prony algorithms by which we got to know the amount of harmonic components which is superimposed in the fundamental waveform to get the final distorted wave.
